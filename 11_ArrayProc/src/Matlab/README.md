# 阵列接收预处理模块 (ArrayProc)

水声通信系统阵列信号处理算法库，覆盖多种阵型配置、DAS/MVDR波束形成、阵元时延标定、空时非均匀变采样重建，含多通道信道仿真和波束方向图可视化。

## 文件清单

| 文件 | 功能 | 类别 |
|------|------|------|
| `gen_array_config.m` | 阵列配置生成（ULA/UCA/自定义） | 配置 |
| `gen_doppler_channel_array.m` | 多通道阵列信道仿真（扩展模块10单通道） | 信道模型 |
| `bf_delay_calibration.m` | 阵元时延标定（互相关法） | 标定 |
| `bf_das.m` | DAS常规波束形成（时延对齐+相干叠加） | 波束形成 |
| `bf_mvdr.m` | MVDR/Capon自适应波束形成 | 波束形成 |
| `bf_nonuniform_resample.m` | 空时联合非均匀变采样重建 | 采样重建 |
| `plot_beampattern.m` | 波束方向图可视化（直角+极坐标） | 可视化 |
| `test_array_proc.m` | 单元测试（11项，含模块10联合） | 测试 |

## 模块功能与接口概述

模块11位于接收链路前端，在模块8'(同步帧解析)之后、模块10-1(粗多普勒补偿)之前。作为可选预处理模块，对下游透明——输出单路高质量复基带信号，下游模块无需改动。

两种工作模式：
- **模式A（精度优先）**：空时变采样重建，等效采样率提升至M·fs，多普勒估计CRLB降低M²倍
- **模式B（SNR优先）**：DAS/MVDR波束形成，SNR提升约10·log₁₀(M) dB

数据流：
- 上游：模块8'(同步帧解析) → **模块11** → 模块10-1(粗多普勒)
- 输入：多通道接收信号 R_array (MxN矩阵，M=阵元数)
- 输出：单路信号 (1xN 或 1x(M·N) 高采样率)

## 支持的阵列类型

| 阵型 | 参数 | 说明 |
|------|------|------|
| ULA（均匀线阵） | 阵元数M、间距d | 沿y轴排列，最常用 |
| UCA（均匀圆阵） | 阵元数M、半径r | xOy平面等角排列，全向覆盖 |
| 自定义 | Mx3坐标矩阵 | 任意三维布局 |

## 波束形成方法对比

| 方法 | 函数 | SNR增益 | 干扰抑制 | 需要先验 | 计算量 |
|------|------|---------|----------|----------|--------|
| DAS | `bf_das` | 10log₁₀(M) dB | 无（固定波束） | 阵元时延 | O(MN) |
| MVDR | `bf_mvdr` | 自适应 | 有（零点对准） | 导向矢量+协方差 | O(M²N+M³) |

## 调用示例

```matlab
% 1. 配置阵列
cfg = gen_array_config('ula', 8, [], 12000);  % 8元ULA，半波长间距

% 2. 生成多通道信道
[R, info] = gen_doppler_channel_array(s, fs, alpha, paths, snr, cfg, theta);

% 3a. DAS波束形成（模式B）
[y_das, gain] = bf_das(R, info.tau_array, fs);

% 3b. MVDR波束形成（模式B，自适应）
a = exp(-1j*2*pi*fc*cfg.positions*look_dir.'/cfg.c);  % 导向矢量
[y_mvdr, w] = bf_mvdr(R, a, 0.01);

% 3c. 非均匀变采样重建（模式A）
[y_hi, eff_fs] = bf_nonuniform_resample(R, info.tau_array, fs);

% 4. 可视化波束方向图
plot_beampattern(cfg, w, 'MVDR 8元ULA');
```

## 函数接口说明

### gen_array_config.m

**功能**：生成阵列配置参数

**输入参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| array_type | 字符串 | 'ula'(默认)/'uca'/'custom' |
| M | 正整数 | 阵元数，默认 8 |
| d | 正实数 | 阵元间距(ULA)或半径(UCA)，默认 λ/2 |
| fc | 正实数 | 载频Hz，默认 12000 |
| varargin | Mx3矩阵 | 自定义阵型的阵元坐标 |

**输出参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| config | 结构体 | 含 `.type`、`.M`、`.positions`(Mx3坐标)、`.d`、`.fc`、`.lambda`、`.c`(1500m/s) |

---

### gen_doppler_channel_array.m

**功能**：多通道阵列信道仿真

**输入参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| s | 1xN 复数 | 发射信号 |
| fs | 正实数 | 采样率 |
| alpha_base | 实数 | 多普勒因子 |
| paths | 结构体 | 多径参数（同gen_doppler_channel） |
| snr_db | 实数 | 信噪比 |
| array_config | 结构体 | 阵列配置（由gen_array_config生成） |
| theta | 实数 | 入射角(弧度)，默认 0 |
| time_varying | 结构体 | 时变参数（可选） |

**输出参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| R_array | MxN_rx 矩阵 | 多通道接收信号 |
| channel_info | 结构体 | 含 `.tau_array`(各阵元时延)、`.theta`、`.array_config` 及单通道信息 |

---

### bf_das.m

**功能**：DAS常规波束形成

**输入参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| R_array | MxN 矩阵 | 多通道接收信号 |
| tau_delays | 1xM 数组 | 各阵元时延补偿量(秒) |
| fs | 正实数 | 采样率 |

**输出参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| output | 1xN 数组 | 波束形成后单路信号 |
| snr_gain | 标量 | SNR提升(dB)，理论值10log₁₀(M) |

---

### bf_mvdr.m

**功能**：MVDR/Capon自适应波束形成

**输入参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| R_array | MxN 矩阵 | 多通道接收信号 |
| steering_vector | Mx1 复数 | 期望方向导向矢量 |
| diag_loading | 正实数 | 对角加载量，默认 0.01 |

**输出参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| output | 1xN 数组 | 波束形成后信号 |
| weights | Mx1 复数 | MVDR权重向量 |

---

### bf_delay_calibration.m

**功能**：阵元时延标定

**输入参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| R_array | MxN 矩阵 | 多通道接收信号 |
| preamble | 1xL 数组 | 已知前导码 |
| fs | 正实数 | 采样率 |
| tau_true | 1xM 数组 | 真实时延（可选，用于计算标定误差） |

**输出参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| tau_est | 1xM 数组 | 估计时延(秒)，第1阵元=0 |
| tau_error | 1xM 数组 | 标定误差(秒)，需提供tau_true |

---

### bf_nonuniform_resample.m

**功能**：空时联合非均匀变采样重建

**输入参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| R_array | MxN 矩阵 | 多通道接收信号 |
| tau_delays | 1xM 数组 | 各阵元精确时延(秒) |
| fs | 正实数 | 原始采样率 |

**输出参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| output | 1x(M·N) 数组 | 重建高采样率信号 |
| effective_fs | 正实数 | 等效采样率(≈M·fs) |

---

### plot_beampattern.m

波束方向图可视化，绘制直角坐标和极坐标方向图。输入：`array_config`(阵列配置)、`weights`(权重，默认等权DAS)、`title_str`(标题)。

---

### test_array_proc.m

单元测试（11项），覆盖阵列配置、多通道信道、DAS/MVDR波束形成、时延标定、方向图、非均匀重建和模块10联合多普勒精度对比。
