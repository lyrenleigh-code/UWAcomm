# 脉冲成形/上下变频模块 (Waveform)

水声通信系统物理层波形处理算法库，覆盖脉冲成形、匹配滤波、数字上下变频、FSK波形生成和DA/AD转换仿真。

## 文件清单

| 文件 | 功能 | 类别 |
|------|------|------|
| `pulse_shape.m` | 脉冲成形（RC/RRC/矩形/高斯，上采样+滤波） | 成形滤波 |
| `match_filter.m` | 匹配滤波（成形滤波器的时间反转共轭） | 成形滤波 |
| `upconvert.m` | 数字上变频（复基带→通带实信号） | 变频 |
| `downconvert.m` | 数字下变频（通带→复基带，含LPF） | 变频 |
| `gen_fsk_waveform.m` | FSK波形生成（频率索引→正弦波形，CPFSK） | 波形生成 |
| `da_convert.m` | DA转换仿真（量化/理想模式） | DA/AD |
| `ad_convert.m` | AD转换仿真（量化/理想模式，含截断） | DA/AD |
| `test_waveform.m` | 单元测试（20项） | 测试 |

## 各功能说明

### 1. 脉冲成形 + 匹配滤波

```matlab
sps = 8; rolloff = 0.35; span = 6;

% 发端成形
[shaped, h, t] = pulse_shape(symbols, sps, 'rrc', rolloff, span);

% 收端匹配
[filtered, ~] = match_filter(received, sps, 'rrc', rolloff, span);

% 下采样提取符号
delay = span*sps/2;
sampled = filtered(delay+1 : sps : end);
```

四种滤波器：

| 类型 | 频域特性 | 适用场景 |
|------|----------|----------|
| RC（升余弦） | 零ISI，Nyquist脉冲 | 理论分析 |
| RRC（根升余弦） | 发收级联=RC，实际系统标配 | PSK/QAM通信 |
| 矩形 | 零阶保持 | 简单系统、OFDM |
| 高斯 | 恒包络，低旁瓣 | GMSK、低PAPR需求 |

### 2. 数字上下变频

```matlab
fs = 48000;   % 采样率 48kHz
fc = 12000;   % 载波频率 12kHz

% 上变频：复基带 → 通带实信号
[passband, t] = upconvert(baseband, fs, fc);

% 下变频：通带 → 复基带（含低通滤波去2fc）
[baseband, t] = downconvert(passband, fs, fc, fc/2);
```

- 上变频：`s(t) = I(t)cos(2πfct) - Q(t)sin(2πfct)`
- 下变频：正交混频 + FIR低通滤波
- 采样率须满足：`fs >= 2*(fc + B/2)`

### 3. FSK波形生成

```matlab
M = 8; f0 = 1000; spacing = 100; fs = 8000; dur = 0.01;
[waveform, t, freqs] = gen_fsk_waveform(freq_indices, M, f0, spacing, fs, dur);
```

- 将MFSK/FH模块的频率索引转为实际正弦波
- 连续相位FSK (CPFSK)：符号边界相位连续
- 正交条件：`freq_spacing >= 1/sym_duration`

### 4. DA/AD转换仿真

```matlab
% DA转换（发端）
[da_out, scale] = da_convert(signal, 12, 'quantize');  % 12bit量化
[da_out, ~]     = da_convert(signal, 16, 'ideal');      % 理想直通

% AD转换（收端）
[ad_out, ~] = ad_convert(received, 14, 'quantize', full_scale);
[ad_out, ~] = ad_convert(received, 16, 'ideal');
```

- 量化模式：均匀量化，SQNR ≈ 6.02×bits + 1.76 dB
- 理想模式：浮点直通，不引入量化噪声
- AD支持指定满量程范围，超出时截断并warning

| 测试 | 断言 | 说明 |

**5. Modulation联合测试（6项）**

基带回环 vs 全链路对比：

| | 基带回环 | 全链路 |
|--|---------|--------|
| 信号域 | 始终在复基带 | 基带→通带→基带 |
| 上下变频 | 无 | 有（cos/sin载波调制 + 正交混频 + LPF） |
| DA/AD | 无 | 有（引入量化噪声） |
| 损失来源 | 仅滤波器边缘截断 | + LPF带通损失 + 量化噪声 + 混频器泄漏 |
| BER预期 | 0（中间段） | 允许少量损失 |

| 测试 | 断言 | 说明 |
|------|------|------|
| 5.1 QPSK全链路 | BER < 15% | 映射→RRC→14bit DA→上变频→AD→下变频→匹配→判决，含DA/AD量化的完整通带回环 |
| 5.2 16QAM基带回环 | 中间段BER=0 | 映射→RRC成形→RRC匹配→AGC→判决，跳过首尾边缘符号验证算法正确性 |
| 5.3 MFSK+FSK波形回环 | 20符号BER=0 | MFSK映射→FSK波形生成→相关频率检测→MFSK解映射，验证FSK波形生成闭环 |
| 5.4 64QAM基带回环 | 中间段BER=0 | 同5.2，64QAM(6bit/符号)高阶调制验证，星座更密，对滤波精度要求更高 |
| 5.5 16QAM全链路 | BER < 15% | 映射→RRC→14bit DA→上变频→14bit AD→下变频→匹配→AGC→判决 |
| 5.6 64QAM全链路 | BER < 15% | 同上，16bit DA/AD + fs=96kHz + sps=16，更高精度保障64QAM的密集星座 |

## 函数接口说明

### pulse_shape.m

**功能**：脉冲成形滤波——将符号序列上采样并进行脉冲成形

**输入参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| symbols | 1xN 复数/实数数组 | 符号序列 |
| sps | 正整数 | 每符号采样数（上采样因子），默认 8 |
| filter_type | 字符串 | 滤波器类型：'rrc'(默认，根升余弦)、'rc'(升余弦)、'rect'(矩形)、'gauss'(高斯) |
| rolloff | 0~1 实数 | 滚降系数（rc/rrc用），默认 0.35。高斯脉冲时为BT积，默认 0.5 |
| span | 正整数 | 滤波器截断长度（符号数），默认 6，总长 = span*sps+1 |

**输出参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| shaped_signal | 1xM 数组 | 成形后的基带信号 |
| filter_coeff | 1xL 数组 | 滤波器系数，归一化为单位能量 |
| t_filter | 1xL 数组 | 滤波器时间轴（单位：符号周期） |

---

### match_filter.m

**功能**：匹配滤波——接收端脉冲成形的匹配操作，最大化SNR

**输入参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| signal | 1xM 复数/实数数组 | 接收基带信号 |
| sps | 正整数 | 每符号采样数，须与发端一致，默认 8 |
| filter_type | 字符串 | 滤波器类型，须与发端一致，默认 'rrc' |
| rolloff | 0~1 实数 | 滚降系数，须与发端一致，默认 0.35 |
| span | 正整数 | 滤波器截断长度，默认 6 |

**输出参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| filtered | 1xM 数组 | 滤波后信号，可下采样提取符号 |
| filter_coeff | 1xL 数组 | 匹配滤波器系数（时间反转共轭的脉冲成形滤波器） |

---

### upconvert.m

**功能**：数字上变频——将复基带信号调制到通带载波频率

**输入参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| baseband | 1xN 复数数组 | 复基带信号 |
| fs | 正实数 | 采样率 (Hz)，须满足 fs >= 2*(fc + B/2) |
| fc | 正实数 | 载波频率 (Hz) |

**输出参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| passband | 1xN 实数数组 | 通带实信号，passband(n) = Re{baseband(n) * exp(j*2*pi*fc*t(n))} |
| t | 1xN 数组 | 时间轴（秒） |

---

### downconvert.m

**功能**：数字下变频——将通带信号解调回复基带

**输入参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| passband | 1xN 实数数组 | 通带实信号 |
| fs | 正实数 | 采样率 (Hz)，须与发端一致 |
| fc | 正实数 | 载波频率 (Hz)，须与发端一致 |
| lpf_bandwidth | 正实数 | 低通滤波器截止频率 (Hz)，默认 fc/2，用于滤除二倍频分量 |

**输出参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| baseband | 1xN 复数数组 | 复基带信号 |
| t | 1xN 数组 | 时间轴（秒） |

---

### gen_fsk_waveform.m

**功能**：FSK波形生成——将频率索引转为实际正弦波形（连续相位CPFSK）

**输入参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| freq_indices | 1xL 数组 | 频率索引序列，取值 0 ~ M-1（由 mfsk_modulate 或 fh_spread 产生） |
| M | 正整数 | 频率数 |
| f0 | 正实数 | 最低频率 (Hz)，默认 1000 |
| freq_spacing | 正实数 | 频率间隔 (Hz)，默认 100。第k个频率 = f0 + k*freq_spacing |
| fs | 正实数 | 采样率 (Hz)，默认 8000 |
| sym_duration | 正实数 | 每符号持续时间 (秒)，默认 0.01 |

**输出参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| waveform | 1xN 实数数组 | FSK时域波形 |
| t | 1xN 数组 | 时间轴（秒） |
| freqs | 1xM 数组 | M个频率值 (Hz) |

---

### da_convert.m

**功能**：DA转换仿真——将浮点信号量化为有限精度（模拟DAC）

**输入参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| signal | 1xN 实数数组 | 输入信号 |
| num_bits | 正整数 | DAC量化比特数，默认 16 |
| mode | 字符串 | 转换模式：'quantize'(默认，均匀量化) 或 'ideal'(直通不做量化) |

**输出参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| output | 1xN 实数数组 | 量化后的信号 |
| scale_factor | 标量 | 归一化缩放因子（用于AD还原） |

---

### ad_convert.m

**功能**：AD转换仿真——将模拟信号量化为数字信号（模拟ADC）

**输入参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| signal | 1xN 实数数组 | 输入模拟信号 |
| num_bits | 正整数 | ADC量化比特数，默认 16 |
| mode | 字符串 | 转换模式：'quantize'(默认) 或 'ideal'(直通) |
| full_scale | 正实数 | ADC满量程范围，默认自动适配信号峰值(+10%余量)。超出 [-full_scale, +full_scale] 时截断 |

**输出参数**：

| 参数 | 类型 | 说明 |
|------|------|------|
| output | 1xN 实数数组 | 量化后的数字信号 |
| scale_factor | 标量 | 满量程值（供后续处理参考） |

---

### plot_eye_diagram.m

眼图绘制——观测脉冲成形后的码间干扰和定时余量。输入：`signal`(基带信号)、`sps`(每符号采样数)、`num_periods`(叠加符号周期数，默认2)、`title_str`(标题)。

---

### test_waveform.m

单元测试脚本（20项），覆盖脉冲成形、上下变频、FSK波形、DA/AD转换、Modulation联合测试和异常输入。
