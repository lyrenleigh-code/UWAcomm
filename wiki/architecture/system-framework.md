## 各模块角色

| 模块 | TX功能 | RX功能 |
|------|--------|--------|
| 01 信源 | 信源编码 | 信源解码 |
| 02 信道编码 | 卷积编码 | SISO(BCJR)译码 |
| 03 交织 | 随机交织 | 随机解交织 + Turbo迭代环 |
| 04 调制 | QPSK映射 | (在07 soft_demapper中) |
| 05 扩频 | DSSS/FH扩频 | 解扩 |
| 06 多载波 | 分块+CP | 去CP+FFT |
| **07 信道估计+均衡** | — | **CIR估计 + BEM跟踪 + 均衡 + 软信息** |
| 08 同步 | gen_lfm+帧组装 | sync_detect+帧解析 |
| 09 波形 | RRC成形+上变频 | 下变频+RRC匹配 |
| 10 多普勒 | — | 估计+重采样补偿 |
| 11 阵列 | — | 波束形成(可选) |
| **12 Turbo调度** | — | **均衡⇌译码迭代调度** |
| 13 集成 | gen_uwa_channel | 端到端仿真 |

## V6 相对 V5 的关键变更

| 项目 | V5 | V6 |
|------|-----|-----|
| 帧信号 | 复基带 | **通带实数**（可DAC输出） |
| LFM前导 | 无 | **gen_lfm通带实数 + 功率归一化** |
| 帧结构 | 数据块无导频 | **[LFM\|训练\|导频\|数据\|导频\|...\|LFM]** |
| 信道估计 | Oracle(ch_info.h_time) | **ch_est_gamp/turbo_vamp（禁止oracle）** |
| 时变信道估计 | 无 | **BEM基扩展 + 散布导频 + DD精化** |
| 同步 | 无 | **sync_detect(直达径窗口) + 有效时延偏移** |
| 多普勒补偿 | comp_resample(-α) | **comp_resample(+α) V7** |
| SC-TDE均衡 | LE(21,0) iter1 | **DFE(31,90)+h_est初始化 iter1** |
| SC-FDE/OFDM Turbo | 单块turbo_equalizer | **跨块crossblock + DD信道更新** |
| Turbo SC-TDE时变 | 固定h_est ISI消除 | **分块LMMSE-IC + BEM每块H_est** |

## 信道估计方法选择指南

```
信道类型判断
    │
    ├─ 静态（fd=0）
    │   └─ ch_est_turbo_vamp（最优）/ ch_est_gamp（推荐）
    │      → 固定h_est → Turbo DFE/LMMSE-IC
    │
    ├─ 慢变（fd≤1Hz）
    │   └─ BEM(Q=5+) + 训练+散布导频(50sym/段)
    │      → 每块H_est重构 → 跨块Turbo LMMSE-IC
    │
    └─ 快变（fd>1Hz）
        └─ BEM(Q=2⌈fd·T⌉+3) + 密集导频
           → 每块H_est重构 → 跨块Turbo LMMSE-IC
           → (可选) DD精化 iter2+
```

## 均衡器选择指南

```
体制判断
    │
    ├─ SC-FDE / OFDM（分块+CP）
    │   └─ eq_mmse_ic_fde + soft_demapper/mapper
    │      → turbo_equalizer_scfde_crossblock（跨块Turbo）
    │
    ├─ SC-TDE 静态
    │   └─ eq_dfe(31,90) + h_est初始化
    │      → turbo_equalizer_sctde V8（DFE iter1 + ISI iter2+）
    │
    ├─ SC-TDE 时变
    │   └─ 分块 + eq_mmse_ic_fde（与SC-FDE相同）
    │      → BEM每块H_est + 跨块Turbo
    │
    └─ OTFS
        └─ eq_otfs_mp / eq_bem_turbo_fde
```
