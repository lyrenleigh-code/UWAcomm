# 模块10 — 多普勒估计与补偿 (DopplerProc)

> 文件夹：`10_DopplerProc/src/Matlab/`
> 框架位置：接收端特有处理，拆分为两个子模块
> - **10-1**：粗多普勒补偿（6'之前）
> - **10-2**：残余多普勒补偿（7'之后）
> 所有体制均使用（OTFS 10-1可弱化）

---

## 功能定位

水声信道的多普勒效应由收发平台相对运动引起，表现为：
- **宽带多普勒**（时间压缩/扩展）：α = v/c，整体信号被压缩/拉伸
- **窄带残余**（载波频偏CFO）：粗补偿后的剩余频率偏移

**10-1 粗多普勒补偿**：在模块6'（去CP+逆变换）之前完成，通过重采样去除整体时间压缩
**10-2 残余多普勒补偿**：在模块7'（均衡）之后完成，校正残余CFO相位旋转

## 两级处理架构

### 10-1 粗多普勒补偿

**位置**：8'(同步) → 11(阵列,可选) → **10-1** → 6'(去CP+逆变换)

**目的**：去除宽带多普勒效应（时间压缩/扩展），确保后续FFT/逆变换不受频偏影响

**流程**：
```
1. 估计多普勒缩放因子 α = v/c
   - CAF: 时延-多普勒2D网格搜索，找峰值
   - xcorr: 复相关幅度估α + 相位估Δf（SC体制）
   - CP: 半符号自相关相位差（OFDM）
2. 重采样补偿
   - 新采样率 fs' = fs × (1+α)
   - 三次样条或Farrow结构插值
```

**各体制差异**：

| 体制 | 估计方法 | 说明 |
|---|---|---|
| SC-TDE | xcorr / CAF | 前导序列互相关 |
| SC-FDE | 复自相关幅相联合 | 利用前/后导码 |
| OFDM | CP自相关 | 半符号相位差 |
| OTFS | **可弱化** | DD域天然分辨多普勒 |
| DSSS | xcorr | 扩频码互相关 |
| FH-MFSK | CAF | 通用方法 |

### 10-2 残余多普勒补偿

**位置**：PTR(可选) → **10-2** → 7'(SISO均衡) 或嵌入均衡器

**目的**：校正粗补偿后的残余CFO（通常10~20Hz级别）

**v5变更**：**10-2移出Turbo迭代环**，在迭代开始前一次性完成

**方法**：
```
残余CFO校正: y_corrected[n] = y[n] × exp(-j·2π·Δf_res·n/fs)
```

**SC-TDE特殊**：10-2的PLL嵌入均衡器(`eq_dfe.m`)内部，与7'紧耦合

## 接口约定

```matlab
% 粗多普勒补偿（一体化调度）
[compensated, alpha_hat] = doppler_coarse_compensate(rx_signal, ref_seq, fs, method)

% 单独估计
alpha = est_doppler_caf(rx_signal, ref_signal, fs, alpha_range, delay_range)
alpha = est_doppler_xcorr(rx_signal, ref_seq, fs)
delta_f = est_doppler_cp(rx_signal, N_fft, cp_len, fs)

% 重采样
resampled = comp_resample_spline(signal, alpha, fs)
resampled = comp_resample_farrow(signal, alpha, fs)

% 残余补偿（一体化调度）
[compensated, cfo_hat] = doppler_residual_compensate(rx_signal, pilot_info, fs)

% CFO旋转
corrected = comp_cfo_rotate(signal, delta_f, fs)
```

## 多普勒效应物理背景

水声信道中的多普勒效应：
- 声速 c ≈ 1500 m/s（远小于电磁波 3×10⁸ m/s）
- 相对速度 v = 1 m/s → α = v/c ≈ 6.67×10⁻⁴
- 对应频偏：Δf = α × fc，若fc=12kHz → Δf ≈ 8Hz
- **水声多普勒效应比射频大5~6个数量级**，必须专门处理

## 与其他模块的关系

```
接收链路：
  [08' 同步] → [11 阵列(可选)] → [10-1 粗多普勒] → [06' 去CP+逆变换]
  → [PTR(可选)] → [10-2 残余多普勒] → [07' SISO均衡(Turbo迭代)]
```

- 10-1必须在06'之前（否则FFT子载波错位/DD域格点偏移）
- 10-2在v5中从Turbo迭代环内移出，只执行一次
- 阵列模块11可提供更高精度的多普勒估计（CRLB降低M²倍）
- SC-TDE的10-2 PLL与07' eq_dfe紧耦合

---

## 备注

- 水声通信中多普勒补偿是必不可少的步骤
- 粗补偿精度目标：残余CFO < 子载波间隔的10%（OFDM/OTFS）
- Farrow结构适合实时FPGA实现，三次样条适合离线仿真
- OTFS的DD域天然处理多普勒是其核心优势之一
- 多普勒仿真信道(`gen_doppler_channel.m`)可模拟时变α(t)场景
