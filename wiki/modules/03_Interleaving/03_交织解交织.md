# 模块03 — 交织/解交织 (Interleaving)

> 文件夹：`03_Interleaving/src/Matlab/`
> 框架位置：发射端第3步 / 接收端解交织
> 所有体制均使用；v5中交织器被纳入Turbo迭代环路

---

## 功能定位

- **发射端**：打散编码后的比特流，使突发错误经解交织后分散为随机错误
- **接收端**：恢复原始比特顺序
- **Turbo均衡（v5）**：交织/解交织成为迭代环路的必要组成部分
  - 3'(解交织)：均衡器外信息 → 译码器
  - 3(交织反馈)：译码器外信息 → 均衡器

## 交织方案详情

### 分组交织（Block）
- 按行写入 R×C 矩阵，按列读出
- 交织深度 = num_rows，决定打散能力
- 突发错误长度 ≤ num_rows 时可完全打散
- 自动补零对齐矩阵尺寸

### 随机交织（Random）
- 基于种子的伪随机置换
- Turbo码标配交织器
- 收发端使用相同种子即可确定性还原
- 打散能力优于分组交织

### 卷积交织（Convolutional）
- B个分支，第i个分支延迟 i×M 个符号
- 总延迟 = (B-1)×M×B 个符号
- 适合流式传输（不需要等待整块数据）
- 解交织器使用互补延迟结构

---

## 接口约定

```matlab
% 分组交织
interleaved = block_interleave(data, num_rows, num_cols)
original    = block_deinterleave(interleaved, num_rows, num_cols)

% 随机交织
[interleaved, perm] = random_interleave(data, seed)
original            = random_deinterleave(interleaved, perm)

% 卷积交织
interleaved = conv_interleave(data, B, M)
original    = conv_deinterleave(interleaved, B, M)
```

## 在Turbo迭代中的角色（v5）

```
┌─── Turbo外信息迭代环路 ───┐
│                             │
│  7'(SISO均衡) → L_e,eq     │
│       ↓                     │
│  3'(解交织) ← 恢复译码器顺序│
│       ↓                     │
│  2'(SISO译码) → L_e,dec    │
│       ↓                     │
│  3(交织反馈) ← 恢复均衡器顺序│
│       ↓                     │
│  soft_mapper → x̄, σ²_x    │
│       ↓                     │
│  → 回到 7'                  │
└─────────────────────────────┘
```

- 交织器打乱了比特顺序，均衡器和译码器工作在不同的比特排列下
- 这种顺序差异正是Turbo均衡能够获得迭代增益的关键——两个SISO模块看到的是"独立"的观测

## 与其他模块的关系

```
[02 信道编码] → [03 交织] → [04 符号映射] → ...
... → [04' 符号判决] → [03' 解交织] → [02' 信道解码]
```

- **上游**：接收模块02编码后的比特流
- **下游**：输出打散后的比特送入模块04映射
- **内部依赖**：`turbo_encode.m` 内部调用随机交织器
- **Turbo均衡**：模块12的迭代调度器在每次迭代中调用交织/解交织

---

## 备注

- 交织深度需要根据信道突发错误长度选择
- 水声信道多径展宽大，建议交织深度 ≥ 最大多径展宽对应的符号数
- Turbo均衡中推荐使用随机交织（与Turbo码交织器一致）

## 模块依赖关系

### 上游（本模块依赖）
- 无（独立工具模块）

### 下游（依赖本模块）
- [[02_信道编解码]] — Turbo编解码内部调用
- [[04_符号映射判决]] — 交织后比特流送调制
- [[12_迭代调度器]] — Turbo均衡迭代环路中的交织/解交织
- [[07_信道估计与均衡]] — 软信息交换中的交织操作

### 在端到端中的位置
- **TX**：[[02_信道编解码]] → `random_interleave` → [[04_符号映射判决]]
- **RX**：[[12_迭代调度器]] SISO均衡输出 → `random_deinterleave` → BCJR译码
- **Turbo环**：均衡 → 解交织 → 译码 → 交织 → 均衡（每次迭代）
