> 框架位置：发射端最后一步(成形+上变频) / 接收端第一步(下变频+匹配滤波)

## 功能定位

- **发射端**：基带信号 → 脉冲成形 → DA转换 → 上变频到通带 → 功放发射

## 脉冲成形滤波器

| 类型 | 说明 | 特性 |
|---|---|---|
| **RRC** (根升余弦) | TX和RX级联 = RC（零ISI） | **推荐**，滚降因子α=0.2~0.5 |
| RC (升余弦) | Nyquist第一准则零ISI | 单端使用 |
| Rect (矩形) | sinc频谱 | 最简单，带外泄漏大 |
| Gauss (高斯) | 无旁瓣，紧凑时域 | GMSK等连续相位调制 |

**RRC成形过程**：
```
TX: 符号 → 上采样(插零) → RRC滤波 → 基带波形
RX: 基带波形 → RRC匹配滤波 → 下采样 → 符号
级联效果: RRC × RRC = RC → 满足零ISI
```

## 上下变频

### 上变频（TX）
```
s_passband(t) = Re{ s_baseband(t) · exp(j·2π·fc·t) }
             = I(t)·cos(2πfct) - Q(t)·sin(2πfct)
```

### 下变频（RX）
```
I(t) = LPF{ r(t) · 2·cos(2πfct) }
Q(t) = LPF{ r(t) · (-2·sin(2πfct)) }
s_baseband(t) = I(t) + j·Q(t)
```

**采样率要求**：fs ≥ 2·(fc + B/2)（满足Nyquist采样定理）

## FSK波形生成

`gen_fsk_waveform.m`：将MFSK频率索引转换为连续相位FSK波形

- CPFSK（连续相位）：避免相位跳变导致的频谱扩展
- 频率间距 ≥ 1/T_sym（正交条件）
- FH-MFSK体制专用

---

## 完整收发链路

```
发射端：
  基带符号 → [RRC脉冲成形+上采样] → [DA转换(14bit)] → [上变频fc] → 通带实信号

接收端：
  通带实信号 → [下变频fc] → [AD转换(14bit)] → [RRC匹配滤波+下采样] → 基带符号
```

---

## 接口约定

```matlab
% 脉冲成形
tx_waveform = pulse_shape(symbols, sps, filter_type, rolloff)
%   sps: 每符号采样数（上采样因子）
%   filter_type: 'rrc' | 'rc' | 'rect' | 'gauss'
%   rolloff: 滚降因子（RRC/RC用）

% 匹配滤波
rx_symbols = match_filter(rx_waveform, sps, filter_type, rolloff)

% 上变频
passband = upconvert(baseband, fc, fs)

% 下变频
baseband = downconvert(passband, fc, fs)

% FSK波形
fsk_waveform = gen_fsk_waveform(freq_indices, M, freq_spacing, fs, sym_duration)

% DAC/ADC
analog = da_convert(digital, num_bits, v_range)
digital = ad_convert(analog, num_bits, v_range)
```

## 与其他模块的关系

```
发射：[08 帧组装] → [09 脉冲成形/上变频] → 【水声信道】
接收：【水声信道】 → [09' 匹配滤波/下变频] → [08' 同步检测]
```

- **发射端最后一步**：将数字基带信号转为模拟通带信号
- **接收端第一步**：将模拟通带信号转回数字基带
- 脉冲成形质量影响ISI和带外泄漏
- 上下变频的载波频率fc决定了声学信号的工作频段

---

## 备注

- 水声通信典型工作频率：数kHz~数十kHz
- RRC滚降因子α越大，时域旁瓣越小但占用带宽越大
- 功放非线性 + 高PAPR → 需要退避(backoff)或PAPR预处理
- 眼图是评估ISI和定时裕度的直观工具
