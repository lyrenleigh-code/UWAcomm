# 模块05 — 扩频/解扩 (SpreadSpectrum)

> 文件夹：`05_SpreadSpectrum/src/Matlab/`
> 框架位置：发射端第5步(可选) / 接收端解扩(可选)
> 仅 DSSS 和 FH-MFSK 体制使用

---

## 功能定位

- **发射端**：用扩频码/跳频图案扩展信号带宽，获得处理增益和抗干扰能力
- **接收端**：相关解扩/解跳频，恢复原始窄带信号

---

## 函数清单

### 扩频码生成（4个）

| 函数文件 | 功能 | 说明 |
|---|---|---|
| `gen_msequence.m` | m序列生成 | LFSR，度数2~15，周期2^n-1 |
| `gen_gold_code.m` | Gold码生成 | m序列对异或，2^n+1个码 |
| `gen_walsh_hadamard.m` | Walsh-Hadamard码 | N×N正交矩阵 |
| `gen_kasami_code.m` | Kasami码生成 | 偶数度数，2^(n/2)+1个码 |

### DSSS直扩（2个）

| 函数文件 | 功能 | 说明 |
|---|---|---|
| `dsss_spread.m` | 直扩扩频 | 符号×扩频码，带宽扩展N倍 |
| `dsss_despread.m` | 直扩解扩 | 相关器累加，处理增益=10log₁₀(N) dB |

### CSK循环移位键控（2个）

| 函数文件 | 功能 | 说明 |
|---|---|---|
| `csk_spread.m` | CSK扩频 | 数据控制m序列循环移位 |
| `csk_despread.m` | CSK解扩 | 多偏移相关峰检测 |

### M-ary码选扩频（2个）

| 函数文件 | 功能 | 说明 |
|---|---|---|
| `mary_spread.m` | M-ary扩频 | log₂(M)比特选择M个正交码之一 |
| `mary_despread.m` | M-ary解扩 | M路相关器，选最大峰 |

### 跳频（2个）

| 函数文件 | 功能 | 说明 |
|---|---|---|
| `gen_hop_pattern.m` | 跳频图案生成 | 伪随机跳频序列 |
| `fh_spread.m` | 跳频扩频 | 按跳频图案偏移频率 |
| `fh_despread.m` | 解跳频 | 逆频率偏移，恢复基带 |

### 非相干检测（2个）

| 函数文件 | 功能 | 说明 |
|---|---|---|
| `det_dcd.m` | 差分相关检测 | DCD，抗频偏 |
| `det_ded.m` | 能量检测 | DED，M路能量比较 |

### 辅助

| 函数文件 | 功能 |
|---|---|
| `plot_code_correlation.m` | 自相关/互相关可视化 |

## 扩频码特性对比

| 码类型 | 码个数 | 自相关旁瓣 | 互相关上界 | 适用场景 |
|---|:---:|:---:|:---:|---|
| m序列 | 1 | -1/N | — | 单用户DSSS |
| Gold码 | 2^n+1 | -1/N | 2^((n+1)/2)+1 | 多用户CDMA |
| Walsh | N | 0(同步) | 高(异步) | 同步CDMA |
| Kasami | 2^(n/2)+1 | -1/N | 低 | 准同步CDMA |

## 接口约定

```matlab
% m序列
seq = gen_msequence(degree)  % degree=2~15, 输出±1

% Gold码
[code1, code2, ...] = gen_gold_code(degree, num_codes)

% DSSS扩频/解扩
spread_signal = dsss_spread(symbols, spreading_code)
despread_signal = dsss_despread(received, spreading_code)
%   处理增益 PG = length(spreading_code)

% 跳频
hop_seq = gen_hop_pattern(num_hops, num_freqs, seed)
spread_signal = fh_spread(signal, hop_seq, freq_spacing)
despread_signal = fh_despread(received, hop_seq, freq_spacing)
```

## 与其他模块的关系

```
[04 符号映射] → [05 扩频] → [07 导频插入] → ...
... → [07' 均衡] → [10-2 残余相位校正] → [05' 解扩] → [04' 判决]
```

- **上游**：接收模块04映射后的符号
- **下游**：扩频后送入模块07导频插入
- DSSS：扩频增益 = 10·log₁₀(码长) dB
- FH-MFSK：跳频增益来自频率分集

---

## 备注

- 扩频的核心优势是处理增益和抗干扰能力，代价是带宽扩展
- 水声信道带宽有限，DSSS的码长通常不宜过大（典型31~127）
- FH-MFSK适合强多径+强干扰场景，但数据速率低
- CSK和M-ary扩频在水声中有特殊应用（高可靠低速率通信）
