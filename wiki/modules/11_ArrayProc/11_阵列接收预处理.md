# 模块11 — 阵列接收预处理 (ArrayProc)

> 文件夹：`11_ArrayProc/src/Matlab/`
> 框架位置：接收端，8'(同步)之后、10-1(粗多普勒)之前
> **可选模块**：需多阵元硬件支持，对下游模块透明

---

## 功能定位

- 利用多阵元空间分集，提升接收信号质量
- 输出单路高质量复基带信号，下游流程无需改动
- 不独立成通信体制，而是作为前处理叠加在任意体制之上

## 两种工作模式

### 模式A — 精度优先（非均匀重采样）

```
M路ADC采样 R[m,n]
    ↓
时延标定 (bf_delay_calibration)
    ↓
空时变采样排序 + 三次样条重建 (bf_nonuniform_resample)
    ↓
等效采样率 ≈ M·fs
    ↓
高精度多普勒估计（CRLB降低 M² 倍）
    ↓
进入标准接收流程
```

**原理**：M个阵元对同一信号的采样时刻不同（因传播时延差），排序后等效于M倍采样率
**优势**：多普勒估计精度 CRLB 降低 M² 倍
**约束**：阵元时延标定误差须 < T_s/(2M)，否则等效采样率提升失效

### 模式B — SNR优先（波束形成）

```
M路ADC采样 R[m,n]
    ↓
DAS波束形成 (bf_das) 或 MVDR (bf_mvdr)
    ↓
SNR提升 ≈ 10·log₁₀(M) dB
    ↓
标准多普勒估计（低SNR场景适用）
    ↓
进入标准接收流程
```

**DAS**：时延对齐+相位补偿+累加，简单鲁棒
**MVDR**：自适应权重，最小方差无失真响应，干扰抑制能力强

## 阵列构型

| 类型 | 说明 | 典型参数 |
|---|---|---|
| ULA (均匀线阵) | M个阵元等间距排列 | d = λ/2 |
| UCA (均匀圆阵) | M个阵元等角度分布在圆上 | 半径 r |
| 自定义 | 任意3D坐标 | Mx3矩阵 |

```matlab
config = gen_array_config('ULA', M, d)
%   config.positions: M×3 坐标矩阵
%   config.M: 阵元数
```

---

## 接口约定

```matlab
% 阵列构型
config = gen_array_config(type, M, spacing)

% 多通道信道仿真
rx_array = gen_doppler_channel_array(tx_signal, config, channel_params)
%   rx_array: M×N 矩阵，M路接收信号

% 时延标定
delays = bf_delay_calibration(rx_array, ref_signal, fs)
%   delays: M×1 各阵元时延估计

% DAS波束形成
bf_output = bf_das(rx_array, delays, fs, steering_vector)
%   bf_output: 1×N 单路输出

% MVDR波束形成
bf_output = bf_mvdr(rx_array, delays, fs, steering_vector)

% 非均匀重采样
[highrate_signal, fs_equiv] = bf_nonuniform_resample(rx_array, delays, fs)
%   fs_equiv ≈ M × fs
```

## 与其他模块的关系

```
[08' 同步检测+帧解析]
    ↓
[11 阵列接收预处理]  ← 插入位置固定
    ↓
[10-1 粗多普勒补偿]
    ↓
后续标准流程...
```

- **插入位置**：同步之后、多普勒之前
- **对下游透明**：输出仍是标准单路复基带信号
- 模式A的高等效采样率可提升10-1粗多普勒估计精度
- 与PTR(模块07)配合：阵列→PTR→均衡，实现空间+时域双重聚焦

## 备注

- 阵列处理是提升水声通信可靠性的重要手段
- 时延标定精度是模式A的瓶颈——标定误差 > T_s/(2M) 则退化
- MVDR在有强干扰源时优于DAS，但需要估计协方差矩阵
- 阵列规模受水声换能器数量和平台尺寸限制
- 模块11的输出维度从M路变为1路，是唯一改变信号维度的模块
