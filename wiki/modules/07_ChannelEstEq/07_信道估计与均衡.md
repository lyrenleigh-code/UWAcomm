- **发射端**：在数据中插入已知导频/训练序列，供接收端估计信道
- **接收端**：
  1. 利用导频估计信道响应

## 核心算法详解

### MMSE-IC频域均衡（v5新建）

`eq_mmse_ic_fde.m` 实现 Tuchler 迭代MMSE-IC：

```
W[k] = H*[k] / (|H[k]|² + σ²_w / σ²_x)
```

- **干扰消除**：Ỹ = Y - (1 - W·H)·X̄（减去已知干扰成分）
- **等效增益**：μ = (1/N)·Σ|H[k]|²/(|H[k]|²+σ²_w/σ²_x)
- **等效噪声方差**：σ²_ñ 用于后续LLR计算
- σ²_x 由 `soft_mapper` 输出，随迭代递减（信心增强→IC更准）

### 软解映射器（v5新建）

`soft_demapper.m`：均衡输出 → 编码比特外信息LLR

```
L_e = 4μ/σ²_ñ · x̃ - L_a    （QPSK简化公式）
```

- 含μ校正：补偿MMSE均衡器的增益偏差
- 先验减除：输出的是外信息，不含先验L_a

### 软映射器（v5新建）

`soft_mapper.m`：后验LLR → 软符号 x̄ + 残余方差 σ²_x

```
x̄ = Σ s_i · P(s_i)     （后验概率加权）
σ²_x = Σ |s_i - x̄|² · P(s_i)
```

- 供 `eq_mmse_ic_fde` 下一轮迭代使用
- σ²_x → 0 表示判决越确定，IC越准

### PTR被动时反转

`eq_ptrm.m`：利用信道估计的时反共轭作为匹配滤波器

```
y_ptr[n] = Σ_m h_m*[-n] ⊛ r_m[n]
```

- 多阵元空间聚焦到主径
- SC-TDE必选，SC-FDE可选
- 等效为空间分集合并+时域均衡预处理

## Turbo均衡数据流（v5完整流程）

```
初始化: x̄=0, σ²_x=1, La_eq=0

for iter = 1:K
  1. [x̃, μ, σ²ñ] = eq_mmse_ic_fde(Y, H, x̄, σ²_x, σ²w)
  2. Le_eq = soft_demapper(x̃, μ, σ²ñ, La_eq)       % 外信息，减去先验
  3. Le_deint = deinterleave(Le_eq)                   % 模块03'
  4. [~, Lpost_info, Lpost_coded] = siso_decode_conv(Le_deint)  % 模块02'
  5. Lpost_int = interleave(Lpost_coded)              % 模块03
  6. [x̄, σ²_x] = soft_mapper(Lpost_int)             % 更新软符号
  7. La_eq = clip(Lpost_int - Le_eq, ±8)             % 新先验 = 后验 - 外信息
end

output: bits = (Lpost_info > 0)    % 最终硬判决
```

## 各体制均衡策略

| 体制 | 信道估计 | 均衡器 | 迭代 |
|---|---|---|---|
| SC-TDE | LS + PTR聚焦 | RLS-DFE+PLL (eq_dfe) | Turbo 3~6次 |
| SC-FDE | 前导码LS+截断 | **MMSE-IC** (eq_mmse_ic_fde) | Turbo 3~6次 |
| DSSS | LS/MMSE | 基本FDE | 无迭代 |
| OFDM | LS+频域插值 | ZF/MMSE FDE | 无迭代（基础） |
| OTFS | DD域稀疏估计 | **MP(BP)** (eq_otfs_mp) | 外层3~5 × 内层10~30 |
| FH-MFSK | LS | 基本均衡 | 无迭代 |

## 与其他模块的关系

```
发射：[04/05/06] → [07 导频插入] → [08 帧组装]
接收：[06' 逆变换] → [PTR] → [10-2 残余CFO] → [07' SISO均衡]

Turbo环路：
  07'(SISO均衡, eq_mmse_ic_fde + soft_demapper)
    → 03'(解交织) → 02'(SISO译码, siso_decode_conv)
    → 03(交织) + soft_mapper → 07'
```

- 与模块02(`siso_decode_conv`)、模块03(交织/解交织)构成Turbo迭代闭环
- 模块12的调度器协调整个迭代流程
