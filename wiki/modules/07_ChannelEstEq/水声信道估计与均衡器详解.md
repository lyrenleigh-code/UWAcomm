## 1. 水声信道基本特性

水声信道（Underwater Acoustic Channel，UAC）是通信领域最具挑战性的物理信道，其核心特点如下：

### 多径传播
声波在海面、海底和水体内部反射，形成多条传播路径。时延扩展可达**几十至几百毫秒**，远大于无线电信道（通常为微秒量级），导致严重的符号间干扰（ISI）。

### 多普勒效应
收发端相对运动（即使仅几节）即可产生显著多普勒效应。水声信道中尤为严重，原因在于声速仅约 **1500 m/s**（约为光速的五百万分之一），相对运动引起的多普勒因子 $a = v/c$ 远大于无线电场景。多普勒不仅表现为简单的频移，更体现为整体的**时间压缩或扩张**。

### 信道时变性
海浪、洋流和温度梯度的不断变化导致信道相干时间短（**几十毫秒至数秒**），是典型的双扩展（时延-多普勒）信道。

### 带宽受限
可用频段通常集中在几 kHz 到几十 kHz，频谱效率的提升十分困难。

### 稀疏性
水声信道虽然时延扩展长，但实际显著多径数目通常远少于时延扩展区间内的采样点数，即**信道冲激响应具有稀疏结构**，这是压缩感知方法的核心利用条件。

## 2. 系统框架概述

一个完整的水声通信系统由以下模块构成：

```
发射端（数据 + 导频）
        │
        ▼
   水声信道（多径·多普勒·噪声·时变）
        │
        ▼
   信道估计（导频辅助 / 盲 / 半盲）
        │
        ▼
   均衡器（ISI 消除·符号恢复）
        │
   ↑（决策反馈）
```

信道估计模块将估计得到的信道冲激响应 $\hat{h}(\tau, t)$ 送入均衡器，均衡器输出判决符号，判决结果又可反馈回信道估计（决策导向模式），形成闭环。

## 3. 信道估计详细步骤

### 3.1 帧结构与导频设计

在发送数据前插入**已知导频序列**，使接收端能够通过已知输入与观测输出推断信道响应。

**常用导频序列：**

| 类型 | 全称 | 主要用途 |
|------|------|----------|
| LFM | 线性调频（Linear Frequency Modulation） | 粗同步、信道探测 |
| PN  | 伪随机序列（Pseudo-random Noise） | 精确信道估计 |
| ZC  | Zadoff-Chu 序列 | 精确估计，恒模低旁瓣 |

**导频插入方式：**
- **块状导频（Block pilot）**：适合慢变信道，帧头集中插入。
- **散布式导频（Scattered pilot）**：适合时变信道，在时频网格上分散插入，密度须满足时间维度的 Nyquist 采样条件（见第6节）。

### 3.2 接收信号同步

利用导频序列与接收信号进行**互相关运算**：

- **定时同步**：找到相关峰位置，确定数据帧的起始时刻，避免 ISI 和 ICI。
- **载波频率同步**：估计载波频率偏差（CFO），对接收信号进行频率校正。

### 3.3 多普勒估计与补偿

水声信道的多普勒不仅是载波频率偏移，更是接收信号在时间轴上的整体**压缩或扩张**，用压缩因子 $a$ 描述：

$$r(t) = s\left(\frac{t}{1+a}\right) * h(t) + n(t)$$

**估计与补偿流程：**

1. 利用宽带导频（如 LFM）估计压缩因子 $\hat{a}$。
2. 对接收信号进行**重采样**（resampling），将时间轴校正回正常速率。
3. 残余多普勒（窄带近似下的频偏分量）再用频域方法补偿。

> **关键区分**：窄带信道中多普勒仅表现为单一频移 $f_d = f_c \cdot v/c$；宽带水声信道中多普勒导致不同频率分量移位量不同，必须用宽带模型处理。

### 3.4 信道冲激响应 (CIR) 估计

多普勒补偿后，信道估计问题可建模为线性方程组：

$$\mathbf{y} = \mathbf{X} \mathbf{h} + \mathbf{n}$$

其中 $\mathbf{X}$ 为导频构成的循环矩阵，$\mathbf{h}$ 为待估 CIR 向量，$\mathbf{n}$ 为噪声。

#### LS 估计（最小二乘）

$$\hat{\mathbf{h}}_{LS} = (\mathbf{X}^H \mathbf{X})^{-1} \mathbf{X}^H \mathbf{y}$$

- 无需信道统计先验，计算简单。
- 高 SNR 下性能良好；低 SNR 下噪声放大严重。
- 复杂度 $O(N^2)$。

#### MMSE 估计（最小均方误差）

$$\hat{\mathbf{h}}_{MMSE} = \mathbf{R}_{hh} \left(\mathbf{R}_{hh} + \sigma_n^2 (\mathbf{X}^H \mathbf{X})^{-1}\right)^{-1} \hat{\mathbf{h}}_{LS}$$

- 利用信道统计先验协方差矩阵 $\mathbf{R}_{hh}$。
- 在噪声抑制与估计精度间取得最优平衡。
- 需要已知信道二阶统计特性，实际中常用近似。
- 复杂度 $O(N^3)$。

#### 压缩感知估计（OMP / SOMP）

利用水声信道的**稀疏多径结构**：实际显著路径数 $K \ll$ 时延区间内采样数 $L$。

**正交匹配追踪（OMP）核心步骤：**

```
初始化：残差 r = y，支撑集 S = {}
循环 K 次：
  1. 找最相关列：k* = argmax |X^H r|
  2. 更新支撑集：S = S ∪ {k*}
  3. LS 估计：h_S = (X_S^H X_S)^{-1} X_S^H y
  4. 更新残差：r = y - X_S h_S
输出：h（仅 S 中元素非零）
```

- 在低导频开销下实现高精度估计（欠定系统求解）。
- 需要信道稀疏度 $K$ 的先验或估计。
- SOMP（Simultaneous OMP）适合多快照联合稀疏恢复。

### 3.5 信道跟踪与更新

信道随时间变化，单次估计无法满足后续符号的需求，需持续更新。

**Kalman 滤波器**（适合已知信道统计特性的平稳时变场景）：

- 状态转移：$\mathbf{h}_n = \mathbf{F} \mathbf{h}_{n-1} + \mathbf{w}$（$\mathbf{F}$ 为信道演化矩阵，$\mathbf{w}$ 为过程噪声）。
- 递推预测与更新，复杂度每步 $O(L^2)$。

**RLS（递推最小二乘）**（无需信道统计先验，遗忘因子 $\lambda$ 控制记忆深度）：

$$J_n = \sum_{k=1}^{n} \lambda^{n-k} |e_k|^2, \quad \lambda \in [0.99, 0.999]$$

**基扩展模型（BEM）**（见第6节详述）：将时变信道参数化后转为静态参数估计问题。

## 4. 均衡器设计

### 4.1 线性均衡器

#### 迫零均衡器（ZF）

$$\mathbf{W}_{ZF} = \mathbf{H}^+$$

直接求信道矩阵的伪逆，完全消除 ISI，但**严重放大噪声**，低 SNR 下性能急剧恶化。

#### MMSE 线性均衡器

$$\mathbf{W}_{MMSE} = \left(\mathbf{H}^H \mathbf{H} + \sigma_n^2 \mathbf{I}\right)^{-1} \mathbf{H}^H$$

在 ISI 消除与噪声放大之间取得平衡。当 $\sigma_n^2 \to 0$ 时退化为 ZF；当 SNR 很低时趋向零强制器。

**适用场景**：时延扩展中等、计算资源受限的场景。

### 4.2 决策反馈均衡器 DFE

DFE 由两部分组成：

- **前馈滤波器（FFF）**：对接收信号滤波，消除前向 ISI（因果方向）。
- **反馈滤波器（FBF）**：利用已判决符号消除后向 ISI（非因果方向）。

均衡器输出：

$$z_n = \mathbf{w}_f^H \mathbf{x}_n - \mathbf{w}_b^H \hat{\mathbf{s}}_{n-1}$$

其中 $\hat{\mathbf{s}}_{n-1}$ 为之前已判决的符号向量。

**核心问题——误差传播**：一旦某符号判决错误，该错误符号进入 FBF，引入额外干扰，可能导致后续连续错误。

**自适应系数更新**（RLS 或 LMS）：
- **LMS**：梯度下降，步长固定，计算简单，收敛慢：$\mathbf{w} \leftarrow \mathbf{w} + \mu \cdot e^* \mathbf{u}$
- **RLS**：逆相关矩阵递推，收敛快，复杂度高：每步 $O(N^2)$

**适用场景**：长时延扩展（如水声信道），比线性均衡器有显著增益。

### 4.3 Turbo 迭代均衡

将均衡器与信道解码器（LDPC / Turbo 码）通过**软信息（对数似然比 LLR）**迭代交换：

```
均衡器 → 软输出 LLR_e → 解码器
解码器 → 外信息 LLR_d → 均衡器（作为先验）
重复 3~5 次迭代
```

每次迭代，均衡器利用解码器提供的符号先验概率消除干扰，解码器则利用更高质量的软信息提升译码精度，双方相互增强。

**关键算法**：
- **BCJR 算法**：在网格图上计算前后向递推，输出每个符号的后验 LLR。
- **MAP 均衡**：最大后验概率准则，复杂度随信道阶数指数增长，实际中常用近似（如 LMMSE 软干扰消除）。

**性能**：3~5 次迭代后性能接近最优 MAP 均衡，是目前水声通信中性能最强的均衡方案之一。

### 4.4 深度学习均衡

以端到端方式训练神经网络，从接收信号直接映射到符号估计，无需显式信道模型。

**常用架构**：
- **BiLSTM**：双向长短期记忆网络，捕捉时序上下文，处理 ISI 效果好。
- **TDNN（时延神经网络）**：多尺度时延特征提取。
- **深度展开（Deep Unfolding）**：将 ISTA、Kalman 等传统迭代算法展开为可学习的神经网络层。

**优势**：适应非高斯噪声、非线性失真，无需精确信道建模。

**局限**：泛化能力依赖训练数据覆盖范围，在训练分布外的信道条件下性能下降。

### 4.5 RLS-DFE 自适应工作流

完整流程分两阶段：

**训练阶段（导频驱动）：**

```
初始化：P = δ⁻¹I，w = 0
对每个导频符号 s_k：
  1. 计算误差：e = s_k - z_k（已知参考符号）
  2. 增益向量：k = P·u / (λ + u^H·P·u)
  3. 更新相关矩阵：P ← (P - k·u^H·P) / λ
  4. 更新权向量：w ← w + k·e*
```

**判决导向阶段（数据期间）：**

```
对每个数据符号位置 n：
  1. FFF 输出：y_f = w_f^H · x_n
  2. FBF 输出：y_b = w_b^H · ŝ_{n-1}
  3. 合并：z_n = y_f - y_b
  4. 硬判决：ŝ_n = slicing(z_n)
  5. 误差：e = ŝ_n - z_n
  6. 同上 RLS 更新 w_f, w_b
```

**参数设置建议**：
- 遗忘因子 $\lambda \in [0.99, 0.999]$（信道越快变，$\lambda$ 应越小）
- 初始化正则化参数 $\delta \in [0.01, 1]$（视 SNR 而定）
- 收敛判断：$\|\mathbf{w}_n - \mathbf{w}_{n-1}\| < \epsilon$

## 5. OFDM 水声通信中的估计与均衡

水声 OFDM 面临的特殊挑战是多普勒破坏子载波正交性，导致载波间干扰（ICI）。

### 关键步骤

**① 导频图案设计**

在时频网格中插入梳状（Comb）或块状（Block）导频，频域做 LS 估计后再插值（DFT 插值或三次样条插值）。

**② 多普勒预补偿**

在 FFT 前对接收信号做时域重采样，消除整体时间压缩因子 $a$：
$$r_{corr}(t) = r((1+\hat{a})t)$$

**③ 残余 ICI 消除**

经过粗补偿后仍存在残余多普勒，造成 ICI。处理方法：
- **串行干扰消除（SIC）**：按子载波信噪比排序，逐一消除干扰。
- **并行干扰消除（PIC）**：一次性减去所有估计干扰，迭代收敛。
- **LMMSE 频域均衡**：最小化包含 ICI 的 MMSE 准则。

**④ 频域单抽头均衡**

补偿后每个子载波做单点乘法：
$$\hat{X}_k = \frac{Y_k}{\hat{H}_k}\ (\text{ZF}) \quad \text{或} \quad \hat{X}_k = \frac{\hat{H}_k^*}{\vert\hat{H}_k\vert^2 + \sigma_n^2/\sigma_s^2} Y_k\ (\text{MMSE})$$

## 6. 时变信道估计的侧重点变化

### 6.1 根本认知转变

| 视角 | 静态/慢变信道 | 时变信道 |
|------|--------------|----------|
| 信道模型 | $h(\tau)$，固定向量 | $h(\tau, t)$，二维函数 |
| 估计目标 | 单次准确恢复 | 持续精确跟踪 |
| 处理方式 | 单快照（single snapshot） | 多快照联合（multi-snapshot） |
| 核心难点 | 低导频下精确恢复 | 跟踪速度 vs 精度权衡 |

**一句话总结**：静态估计的核心词是**"精度"**，时变估计的核心词是**"时效性"**。再精确的估计，如果滞后于信道变化，送到均衡器时已经失效，反而不如一个粗糙但及时的跟踪结果。

### 6.2 三大新增侧重

**① 时间相关性建模（新增核心问题）**

静态估计无需关心信道如何随时间变化；时变估计必须对时间演化建立模型，常用方法：

- **自回归模型（AR）**：$h_n = \sum_{p=1}^{P} a_p h_{n-p} + w_n$，适合平稳时变过程。
- **马尔可夫模型**：一阶马尔可夫 $h_n = F h_{n-1} + w_n$，是 Kalman 滤波的基础。
- **基扩展模型（BEM）**：将时变参数化为基函数线性组合（见 6.3 节）。

**② 导频密度与时间对齐（新增约束）**

时变信道要求导频在时间维度上满足采样定理：

$$f_{pilot} \geq 2 f_{D,max}$$

其中 $f_{D,max}$ 是最大多普勒频率。违反此条件将导致信道时间变化发生混叠，跟踪失效。

同时，散布式导频需要精确的时间对齐，以确保每个导频位置的信道值能够正确映射到时频网格坐标。

**③ 预测-跟踪-更新闭环（新增系统结构）**

静态估计是开环的（估计一次即完成）；时变估计必须构建闭环：

```
估计（当前帧） → 预测（下一帧） → 观测校正 → 更新 → 循环
```

### 6.3 基扩展模型 BEM

BEM 将时变 CIR 展开为一组时间基函数的线性组合：

$$h(\tau, t) = \sum_{q=0}^{Q-1} c_q(\tau) \cdot b_q(t)$$

**常用基函数：**

| 类型 | 基函数 $b_q(t)$ | 特点 |
|------|----------------|------|
| CE-BEM | $e^{j2\pi q \Delta f t}$（复指数） | 建模精确，但存在 Gibbs 现象 |
| P-BEM  | $t^q$（多项式） | 适合短观测窗口，低阶平滑 |
| DCT-BEM | 离散余弦基 | 边界效应小，能量集中 |
| OBE-BEM | 正交多项式 | 最小建模误差，计算稍复杂 |

**优势**：将无限维时变信道压缩为有限维系数 $\{c_q(\tau)\}$ 的估计，转化为标准线性估计问题。

**关键参数**：BEM 阶数 $Q$，选择准则通常为 $Q \geq \lceil 2 f_{D,max} T_{block} \rceil + 1$，其中 $T_{block}$ 为处理数据块长度。

### 6.4 Kalman 滤波器预测-更新循环

Kalman 滤波器是时变信道跟踪的标准工具，以状态空间模型为基础：

**状态方程（信道演化模型）：**

$$\mathbf{h}_n = \mathbf{F} \mathbf{h}_{n-1} + \mathbf{w}_n, \quad \mathbf{w}_n \sim \mathcal{CN}(\mathbf{0}, \mathbf{Q})$$

**观测方程（导频观测）：**

$$\mathbf{y}_n = \mathbf{X}_n \mathbf{h}_n + \mathbf{v}_n, \quad \mathbf{v}_n \sim \mathcal{CN}(\mathbf{0}, \mathbf{R})$$

**预测步：**

$$\hat{\mathbf{h}}_{n|n-1} = \mathbf{F} \hat{\mathbf{h}}_{n-1|n-1}$$

$$\mathbf{P}_{n|n-1} = \mathbf{F} \mathbf{P}_{n-1|n-1} \mathbf{F}^H + \mathbf{Q}$$

**更新步：**

$$\boldsymbol{\nu}_n = \mathbf{y}_n - \mathbf{X}_n \hat{\mathbf{h}}_{n|n-1} \quad \text{（新息）}$$

$$\mathbf{K}_n = \mathbf{P}_{n|n-1} \mathbf{X}_n^H \left(\mathbf{X}_n \mathbf{P}_{n|n-1} \mathbf{X}_n^H + \mathbf{R}\right)^{-1}$$

$$\hat{\mathbf{h}}_{n|n} = \hat{\mathbf{h}}_{n|n-1} + \mathbf{K}_n \boldsymbol{\nu}_n$$

$$\mathbf{P}_{n|n} = (\mathbf{I} - \mathbf{K}_n \mathbf{X}_n) \mathbf{P}_{n|n-1}$$

**物理意义**：Kalman 增益 $\mathbf{K}_n$ 在"相信预测"与"相信观测"之间自动权衡。当观测噪声大时，$\mathbf{K}_n$ 小，倾向于依赖预测；当预测不确定性大时，$\mathbf{K}_n$ 大，倾向于依赖观测。

### 6.5 主要算法对比

| 算法 | 建模能力 | 计算复杂度 | 先验需求 | 适用场景 |
|------|----------|-----------|----------|----------|
| Kalman 滤波 | 线性高斯时变 | 中，$O(L^2)$/步 | 需 F, Q, R | 平稳时变，已知统计特性 |
| RLS 自适应 | 无明确信道模型 | 中，$O(N^2)$/步 | 遗忘因子 $\lambda$ | 无先验，在线自适应 |
| BEM + LS/MMSE | 带限时变 | 低-中 | BEM 阶数 $Q$ | 多普勒扩展已知 |
| SAGE / EM | 多径参数精估 | 高，迭代 | 路径数 $K$ | 高分辨率参数估计 |
| T-SBL（时序稀疏贝叶斯） | 稀疏 + 时变 | 高 | 稀疏先验 | 稀疏且快速时变 |
| 深度展开 / LSTM | 非线性，数据驱动 | 推断低（训练高） | 训练数据 | 非高斯噪声，难建模场景 |
| 粒子滤波 | 非线性非高斯 | 很高，$O(N_{particles})$ | 提议分布 | 非线性演化，性能要求极高 |
