# 模块01 — 信源编解码 (SourceCoding)

> 文件夹：`01_SourceCoding/src/Matlab/`
> 框架位置：发射端第1步 / 接收端最后一步
> 所有体制均使用

---

## 功能定位

- **发射端**：对原始数据进行压缩编码，减少冗余，降低传输比特率
- **接收端**：信源解码，还原原始数据 → BER / FER 评估

---

## 函数清单

| 函数文件 | 功能 | 说明 |
|---|---|---|
| `huffman_encode.m` | Huffman编码 | 变长无损压缩，根据符号概率构建最优前缀码 |
| `huffman_decode.m` | Huffman解码 | 利用码树还原原始符号序列 |
| `uniform_quantize.m` | 均匀量化 | L=2^bits 级均匀量化，中点策略 |
| `uniform_dequantize.m` | 均匀反量化 | 根据量化参数还原模拟值 |
| `test_source_coding.m` | 测试脚本 | 14个测试用例 |

## 编码方案

### Huffman编码
- 变长编码，基于符号出现概率构建二叉树
- 高频符号用短码，低频符号用长码
- 平均码长逼近信源熵 H(X)
- 适用于离散信源的无损压缩

### 均匀量化
- 将连续信号映射到 L = 2^bits 个离散电平
- 量化步长 Δ = (max - min) / L
- 量化噪声功率 σ²_q = Δ²/12
- SQNR ≈ 6.02 × bits + 1.76 dB

---

## 接口约定

```
% 编码
[encoded_bits, codebook] = huffman_encode(symbols, probabilities)

% 解码
decoded_symbols = huffman_decode(encoded_bits, codebook)

% 量化
[indices, params] = uniform_quantize(signal, num_bits)

% 反量化
signal_hat = uniform_dequantize(indices, params)
```

## 与其他模块的关系

```
信源数据 → [01 信源编码] → [02 信道编码] → ...
... → [02' 信道解码] → [01' 信源解码] → BER/FER评估
```

- **上游**：原始数据输入
- **下游**：输出二进制比特流送入模块02信道编码
- 接收端信源解码后进行误码率统计

---

## 备注

- 在水声通信仿真中，信源编码常被简化或跳过（直接生成随机比特）
- 实际系统中可选用更高效的压缩方案（如算术编码）
- 量化主要用于模拟信号数字化场景
