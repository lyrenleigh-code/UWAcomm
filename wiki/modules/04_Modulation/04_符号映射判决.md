# 模块04 — 符号映射/判决 (Modulation)

> 文件夹：`04_Modulation/src/Matlab/`
> 框架位置：发射端第4步(映射) / 接收端判决
> 所有体制均使用，映射方案因体制而异

---

## 功能定位

- **发射端**：将编码交织后的比特映射为复数星座符号（或频率索引）
- **接收端**：对均衡后的符号进行硬判决或软判决（输出LLR）

---

## 函数清单

| 函数文件 | 功能 | 说明 |
|---|---|---|
| `qam_modulate.m` | QAM调制 | 支持BPSK/QPSK/8QAM/16QAM/64QAM |
| `qam_demodulate.m` | QAM解调 | 硬判决或Max-Log-MAP软判决(LLR) |
| `mfsk_modulate.m` | MFSK调制 | 2/4/8/16-FSK 频率索引映射 |
| `mfsk_demodulate.m` | MFSK解调 | 能量检测判决 |
| `plot_constellation.m` | 星座图绘制 | 含比特标注的星座图 |
| `test_modulation.m` | 测试脚本 | 25个测试用例 |

## 调制方案详情

### QAM族（PSK/QAM）

| 调制方式 | 每符号比特数 | 典型应用 |
|---|:---:|---|
| BPSK | 1 | DSSS，低SNR场景 |
| QPSK | 2 | SC-TDE/SC-FDE 常用 |
| 8-QAM | 3 | 中等速率 |
| 16-QAM | 4 | OFDM/OTFS 高速率 |
| 64-QAM | 6 | 高SNR高速率 |

**特性**：
- Gray映射（格雷码）：相邻星座点仅差1bit，BER优于自然映射
- 归一化：平均符号功率为1
- 软判决LLR：Max-Log-MAP近似

$$LLR_k = \frac{1}{\sigma^2}\left(\min_{s:b_k=0}|y-s|^2 - \min_{s:b_k=1}|y-s|^2\right)$$

### MFSK（多频移键控）

| M值 | 每符号比特数 | 频率数 |
|---|:---:|:---:|
| 2-FSK | 1 | 2 |
| 4-FSK | 2 | 4 |
| 8-FSK | 3 | 8 |
| 16-FSK | 4 | 16 |

**特性**：
- 非相干检测，不需要载波相位信息
- 频率索引映射：比特→频率编号(0~M-1)
- 能量检测(DED)或差分相关检测(DCD)
- FH-MFSK体制专用

## 接口约定

```matlab
% QAM调制
symbols = qam_modulate(bits, M, 'gray')  % M=2(BPSK),4(QPSK),16,64...
%   symbols: 复数列向量，|E[|s|²]|=1

% QAM解调（软判决）
llr = qam_demodulate(received, M, 'soft', noise_var)
%   llr: 每比特LLR，正值→bit=0，负值→bit=1

% QAM解调（硬判决）
bits = qam_demodulate(received, M, 'hard')

% MFSK调制
freq_indices = mfsk_modulate(bits, M)

% MFSK解调
bits = mfsk_demodulate(freq_indices, M)
```

## 各体制映射选择

| 体制 | 调制方式 | 说明 |
|---|---|---|
| SC-TDE | QPSK / QAM | PSK/QAM均可 |
| SC-FDE | QPSK / QAM | 常用QPSK |
| DSSS | BPSK | 扩频前通常用BPSK |
| OFDM | QAM | 16-QAM / 64-QAM |
| OTFS | QAM | DD域映射，常用4/16-QAM |
| FH-MFSK | **MFSK** | 频率索引，非星座映射 |

## 与其他模块的关系

```
[03 交织] → [04 符号映射] → [05 扩频(可选)] / [06 多载波(可选)] / [07 导频插入]
... → [07' 均衡] → [04' 符号判决] → [03' 解交织] → ...
```

- **上游**：接收模块03交织后的比特流
- **下游**：输出复数符号送入模块05/06/07
- **Turbo均衡**：判决输出LLR送入解交织→SISO译码器
- **soft_demapper.m**（模块07）在Turbo迭代中承担了更精细的软解映射功能

---

## 备注

- Gray映射是默认推荐，BER性能优于自然映射约0.5~1dB
- 软判决(LLR)比硬判决约有2dB编码增益
- 高阶QAM对信道估计和均衡精度要求更高
- MFSK天然抗多径（非相干检测），但频谱效率低
